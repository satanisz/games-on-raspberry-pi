# Aikido

Prosta przeglądarkowa bijatyka: Staś kontra zły Sensei.

## Uruchomienie lokalne

Gra jest statyczną częścią frontendu i po `npm run build` trafia pod `/aikido/`.
Do samodzielnego podglądu można też otworzyć `index.html` w nowoczesnej przeglądarce.

## Sterowanie

- `←` / `→` — ruch
- `↑` — skok
- `Spacja` — cios ręką
- `X` — kopnięcie

Na telefonie użyj pięciu dużych przycisków pod areną. Sterowanie obsługuje kilka palców jednocześnie, więc można poruszać się i atakować w tym samym czasie.

Przeciwnicy zmieniają się po zwycięstwach: raz walczysz ze złym Sensei, a raz z Agentem. Nagrody i liczba zwycięstw zapisują się automatycznie w pamięci przeglądarki.
