# Kurka Wodna

Oryginalna przeglądarkowa strzelnica zręcznościowa nad stawem.

## Zasady

- klikaj lub dotykaj kolorowych kurek,
- złota kurka daje najwięcej punktów,
- biały łabędź odejmuje punkty i zeruje combo,
- runda trwa 45 sekund, a rekord zapisuje się w pamięci przeglądarki.

Gra jest statyczną częścią frontendu i po `npm run build` trafia pod `/kurka-wodna/`.
